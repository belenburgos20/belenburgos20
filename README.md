## 🧩 Proyecto Destacado

### 🏢 Oleohidráulica Guardese — Sistema Web Full-Stack
Aplicación web desarrollada como proyecto final de la Tecnicatura en Programación.

**Características principales:**
- API REST con Node.js, Express y TypeScript
- Autenticación JWT
- Base de datos PostgreSQL con Sequelize
- Frontend en React + TypeScript + Vite
- Portal de clientes y panel administrativo
- Deploy en producción (Render)

👉 Enfocado en arquitectura escalable, trabajo en equipo y buenas prácticas.
